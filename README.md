# Dynamic constra template

"Constra" is an HTML template developed by themefisher.com.

![alt text](https://raw.githubusercontent.com/adnanafzal565/images/main/Converting%20constra%20template%20into%20dynamic%20website%20-%20Vue%20JS%20%2B%20Laravel.png)

We have also created a tutorial series on this. In [first part](https://youtu.be/dCfKW5PpvAE), you will learn:

- How to convert an HTML template into a Vue JS app.
- Create single-page applications in Vue JS.
- Use the vue-router module.
- Use internal CSS, JS, and image files from the "assets" folder.
- How to fix the "Mixed spaces and tabs" error.
- Remove ESLint from the Vue JS app
- Use Bootstrap, jQuery, and other JS libraries in the Vue JS app.
- How to fix "Can't resolve popper.js"
- Use "background-image" property in Vue JS
- Use slider library (Shuffle) in Vue JS

I am converting this static template into a dynamic website using Vue JS as frontend and Laravel as backend.

You can download the official "constra" template from themefisher.com: https://themefisher.com/products/constra
